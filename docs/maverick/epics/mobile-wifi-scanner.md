# Epic: mobile-wifi-scanner

Owns the Astra mobile client and mobile/iOS scanning feasibility.

## Goal

Make the mobile product experience useful for home network users while understanding which scanning capabilities can run directly from a phone.

## Scope

- Expo / React Native app behavior.
- Mobile UX for device discovery, labels, and insights.
- Phone-to-agent connection flows.
- iPhone-only feasibility research.

## Out Of Scope

- Desktop Go scanner internals except where mobile depends on them.
- Authenticated router-admin ingestion.
