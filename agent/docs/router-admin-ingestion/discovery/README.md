# Discovery Notes

Use this folder for dated discovery notes that future workstreams can continue from without reopening old Discord threads.

## File naming

- `YYYY-MM-DD-<target>-<topic>.md`

## Each note should capture

- what page or flow was inspected
- whether the browser session or shell was used
- what artifacts were produced and where they were saved
- what was confirmed
- what still blocks implementation
- the exact next technical step
